# ✅ To Do List App

A simple and clean **To Do List application** built using **HTML, CSS, and JavaScript**.  
The app allows users to add, update, delete, and clear tasks with data saved in **LocalStorage**.

---

## 🚀 Features

- ➕ Add new tasks
- ✏️ Update existing tasks
- ❌ Delete single task
- 🧹 Clear all tasks
- 💾 Save tasks in browser LocalStorage
- 📱 Responsive and clean UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
